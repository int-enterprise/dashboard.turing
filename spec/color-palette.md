# Color Palette

프로젝트 컬러 팔레트 정의. 각 색상은 100~900까지 9단계의 shade로 확장한다.
브랜드 원본 색상은 각 색상군의 "anchor" 단계에 배치한다.

---

## 사용 비율

| 분류              | 사용 비율 | 용도                                            |
| ----------------- | --------- | ----------------------------------------------- |
| Primary / Base    | 60%       | 본문, 배경, 기본 UI                             |
| Secondary 1, 2, 3 | 30%       | 다이어그램, 플로우차트, 인포그래픽              |
| Accent            | 10%       | 강조, 액션, 하이라이트                          |

원본 3색 (`#FFFFFF`, `#FAFAFA`, `#F4F7FA`)을 100/200/300에 배치하고 상위 단계는 중간 톤으로 확장.
